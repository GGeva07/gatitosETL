## ?? GLOSARIO DE TÉRMINOS
???????????????????????????????????????????????????????????????????????????????

ETL
?? Extract, Transform, Load
?? Proceso de lectura, transformación y carga de datos

DIM / DIMENSION
?? Tabla de dimensión en data warehouse
?? Ej: DimCiudad, DimPersona

ASYNC/AWAIT
?? Patrón de programación asincrónica
?? No bloquea threads durante I/O

GENERIC REPOSITORY
?? Patrón genérico que funciona con cualquier tipo
?? Reutilizable para todas las entidades

IDENTITY
?? Autoincremento en SQL Server
?? Genera IDs automáticamente

FK (FOREIGN KEY)
?? Clave foránea
?? Relación entre tablas

DI (DEPENDENCY INJECTION)
?? Inyección de dependencias
?? Container resuelve dependencias automáticamente
