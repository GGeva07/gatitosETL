## DESCRIPCIÓN GENERAL

El sistema ETL (Extract, Transform, Load) gatitosETL es una aplicación .NET 8
desarrollada como Worker Service que automatiza el proceso de lectura,
transformación y carga de datos desde archivos CSV a una base de datos
dimensional (data warehouse) en SQL Server.

???????????????????????????????????????????????????????????????????????????????
## ?? OBJETIVO

Procesar archivos CSV con información sobre personas, gatos y ciudades,
normalizando los datos en una estructura dimensional optimizada para análisis.

## ?? FLUJO PRINCIPAL

1. LECTURA (Extract)
   ?? Lee archivo CSV: data/datos.csv
   ?? Parsea líneas respetando formato
   ?? Retorna List<CsvDataDto>

2. TRANSFORMACIÓN (Transform)
   ?? Procesa ciudades ? DimCiudad
   ?? Procesa personas ? DimPersona (vinculadas con ciudades)
   ?? Procesa gatos ? DimGato
   ?? Procesa fechas ? DimFecha (descompuestas día/mes/ańo)

3. CARGA (Load)
   ?? INSERT en DimCiudades (5 registros)
   ?? INSERT en DimPersonas (5 registros)
   ?? INSERT en DimGatos (5 registros)
   ?? INSERT en DimFechas (5 registros)

## ?? SERVICIOS PRINCIPALES

1. CsvReaderService (ICsvReaderService)
   ?? Responsabilidad: Leer y parsear archivos CSV
   ?? Método: ReadCsvAsync(filePath) ? List<CsvDataDto>
   ?? Características:
      ?? Parser robusto con manejo de comillas
      ?? Trim de espacios en blanco
      ?? Manejo de excepciones

2. EtlService (IEtlService)
   ?? Responsabilidad: Orquestar transformación y carga
   ?? Método: ProcessEtlAsync(csvData) ? EtlResultDto
   ?? Características:
      ?? Procesamiento secuencial (orden de dependencias)
      ?? Validación de tipos (int.TryParse, DateTime.TryParse)
      ?? Detección de duplicados
      ?? Manejo de relaciones (FK)
      ?? Estadísticas de procesamiento

3. Servicios de Negocio
   ?? CiudadService: CRUD + búsquedas
   ?? PersonaService: CRUD + búsquedas complejas
   ?? GatoService: CRUD + filtros
   ?? FechaService: CRUD + rangos

## ?? ENTIDADES Y RELACIONES

DimCiudad
?? id_ciudad: INT (PK, IDENTITY)
?? nombre: NVARCHAR(255) (UNIQUE)

DimPersona
?? id_persona: INT (PK)
?? nombre: NVARCHAR(255)
?? idCiudad: INT (FK ? DimCiudad)

DimGato
?? id_gato: INT (PK)
?? nombre: NVARCHAR(255)
?? raza: NVARCHAR(100)
?? tipo: NVARCHAR(100)

DimFecha
?? id_fecha: INT (PK, IDENTITY)
?? dia: INT (1-31)
?? mes: INT (1-12)
?? anio: INT (> 1900)
