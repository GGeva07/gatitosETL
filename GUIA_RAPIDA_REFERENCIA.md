## ?? SOLUCI”N DE PROBLEMAS

PROBLEMA: "Connection timeout"
SOLUCI”N:
  1. Verifica SQL Server est· corriendo
  2. Verifica nombre del servidor en appsettings.json
  3. Verifica que la BD existe: SELECT * FROM master.dbo.sysdatabases

PROBLEMA: "The type or namespace name could not be found"
SOLUCI”N:
  1. dotnet restore
  2. dotnet clean
  3. dotnet build

PROBLEMA: "File not found"
SOLUCI”N:
  1. Verifica que datos.csv existe en WorkerService/data/
  2. Verifica la ruta exacta en Worker.cs

PROBLEMA: "No se insertan registros"
SOLUCI”N:
  1. Verifica conexiÛn a BD
  2. Ejecuta: SELECT COUNT(*) FROM DimCiudades;
  3. Verifica logs en consola para errores

PROBLEMA: "Duplicados se insertan"
SOLUCI”N:
  1. Ejecuta: DELETE FROM DimPersonas; DELETE FROM DimCiudades;
  2. Ejecuta el ETL nuevamente
