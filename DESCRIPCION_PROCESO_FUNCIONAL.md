## ?? VISIÓN GENERAL DEL PROCESO

El sistema ETL (Extract, Transform, Load) es un proceso automático que:
1. Lee datos de un archivo CSV
2. Transforma y valida los datos
3. Los carga en la base de datos

Todo esto ocurre de forma **asincrónica** y con **manejo robusto de errores**.

## ?? DETALLES TÉCNICOS DE ASYNC/AWAIT
???????????????????????????????????????????????????????????????????????????????

El sistema es completamente asincrónico:

1. ReadCsvAsync() ? usa File.ReadAllLinesAsync()
2. ProcessEtlAsync() ? espera todas las fases
3. FindAsync() ? consulta base de datos sin bloquear
4. GetByIdAsync() ? búsqueda no bloqueante
5. AddAsync() ? inserción sin bloquear hilo
6. SaveChangesAsync() ? persiste cambios sin bloquear

Ventajas:
? No bloquea recursos
? Mejor rendimiento
? Mejor escalabilidad
? Puede procesar múltiples archivos simultáneamente
